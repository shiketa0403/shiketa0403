# チャンネル個別情報ナレッジ

このディレクトリには、各チャンネル固有の情報をユーザーが手動で記録するファイルを置きます。AIが訓練データやWeb検索で拾えない（あるいは精度が低い）以下のような情報を補うためのものです。

## ファイル名規則

`{slug}.md` （例: `m-on.md`, `wowow.md`, `j-sports.md`）

スラッグはスプレッドシートの「URLスラッグ案」列に対応します。

## 用途

各チャンネル記事生成時、Step 1（ペルソナ分析）・Step 2（見出し構成）・Step 4（本文作成）の補足情報として **CHANNEL_NOTES** 変数に注入されます。

## 推奨フォーマット

`_template.md` を見本としてください。空欄でも問題ありませんが、以下の項目をなるべく埋めると差別化のある記事になります。

- 看板番組・主要コンテンツ
- 対象視聴者層（年齢・性別・趣向）
- 類似チャンネルとの違い
- そのチャンネルでしか取り扱わない番組ジャンル
- 過去の話題コンテンツ・キャンペーン
- ユーザー側の独自視点（実際に視聴した感想など）

## 運用ルール

- ファイルが存在しない・空のチャンネルは、AIが汎用的な情報のみで記事化する
- 重要な記事（CV単価が高いチャンネル）から優先的に書き込む
- 内容に変動があれば随時更新（例：番組名称変更、看板番組の終了など）
